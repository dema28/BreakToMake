## TC_MMV_MODUL_034 — Проверка отображения и функционирования сайта в режиме инкогнито (Firefox Browser)

### Description / Описание
    -

### Preconditions / Предусловия
    Открыт Firefox Browser: 136.0.2 в режиме инкогнито
    Сайт доступен по адресу https://modultest1.framer.website

### Test Steps / Шаги тестирования
    1. Открыть Firefox Browser в режиме инкогнито
    2. Перейти на сайт https://modultest1.framer.website
    3. Проверить загрузку главной страницы
    4. Перейти в раздел с каталогом домов
    5. Перейти на вкладку "Kontakty"
    6. Открыть форму обратной связи
    7. Проверить доступность и кликабельность всех основных элементов

### Expected Result / Ожидаемый результат
    1. Браузер запущен в режиме инкогнито
    2. Страница сайта отобразилась
    3. Главная страница отображается
    4. Страница с каталогом домов отобразилась
    5. Страница с контактами отображается
    6. Форма обратной связи активна и принимает данные
    7. Основные элементы кликабельны и доступны

### Actual Result / Фактический результат
